# Política de Privacidade — My Finance

Última atualização: 2026-02-25

O **My Finance** foi projetado para funcionar em modo local (offline-first).

## 1) Dados coletados
- O aplicativo **não exige criação de conta**.
- As transações e preferências do usuário são armazenadas localmente no dispositivo (AsyncStorage/SecureStore).
- Não há coleta obrigatória de dados sensíveis para uso básico.

## 2) Uso dos dados
- Os dados locais são usados apenas para exibir o histórico financeiro e cálculos de saldo no próprio app.
- Quando o usuário configura um backend próprio, as informações podem ser enviadas para esse endpoint escolhido pelo usuário.

## 3) Compartilhamento
- O app não compartilha dados com terceiros por padrão.
- Não usamos SDK de anúncios no build atual.

## 4) Segurança
- Sessão/token local é armazenada de forma segura via SecureStore quando aplicável.
- Recomendamos proteger o dispositivo com senha/biometria.

## 5) Exclusão de dados
- O usuário pode apagar os dados ao desinstalar o app ou limpando dados do aplicativo no Android.

## 6) Crianças
- O app não é direcionado especificamente para crianças.

## 7) Contato
- E-mail de suporte/publicação: **suporte@myfinance.app**

## 8) Alterações desta política
- Esta política pode ser atualizada para refletir novas funcionalidades.
